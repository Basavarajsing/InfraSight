InfraSight is a lightweight observability dashboard for infrastructure telemetry, combining live log/monitoring views with quick incident-oriented analysis.

## Domain

- Observability Engineering
- Site Reliability Engineering (SRE)
- DevOps Monitoring

## Tech Stack

- Next.js (App Router)
- TypeScript
- API Routes
- Recharts

## Deployment

This project is ready for Vercel deployment.

- Next.js is auto-detected by Vercel.
- No extra Vercel configuration is required for standard deployment.

## Run Locally

1. Install dependencies:

	```bash
	npm install
	```

2. Start the development server:

	```bash
	npm run dev
	```

3. Open:

	```
	http://localhost:3000
	```

## Features: Live Monitoring + Upload Analysis

- **Live monitoring:** The app provides a live stream/preview workflow for observing infrastructure signals in near real time.
- **Upload analysis:** You can upload log-like inputs for API-driven parsing and analysis to surface useful diagnostics quickly.
